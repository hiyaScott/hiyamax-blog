🪁 纸鸢归 🥮 外婆的月饼模 🚗 平行停车
构思 → 角色 → 分镜 → 生成 → 剪辑 → 发布
1
故事构思
从想法到剧本，确定核心创意
📥 设计输入

原始想法：

想要做一个关于中国传统文化的温情短片，弥补之前作品中缺少的中式家庭题材空白。

参考作品：

• 《末世微光》的简单核心关系
• 《时间银行》的双向牺牲结构
• 《记忆典当铺》的记忆可视化

🤔 决策记录
选择纸鸢文化 阿尔茨海默症主题 祖孙二人关系

决策理由：

1. 纸鸢是非遗文化，有视觉辨识度

2. 阿尔茨海默症是现代社会议题

3. 祖孙互动天然具有情感张力

✅ 成果输出

《纸鸢归》故事核心：

一位患阿尔茨海默症的祖父，每天制作纸鸢却记不清为什么。孙子发现祖父是在完成20年前的承诺——每年为孙子做一只纸鸢，直到第100只。

四幕结构：

建立 → 追寻 → 真相 → 传承

2
角色设计
确定视觉风格，设计角色提示词
📥 设计输入

角色需求：

• 祖父：70岁左右，手艺人气质，慈祥但略带迷茫
• 孙子：8-10岁，活泼懂事，与祖父有深厚感情

风格方向：

写实渲染 + 温暖色调 + 中国传统元素

🤔 决策记录
写实风格 竹青色服装 工作间场景

提示词迭代：

v1: 过于卡通化 ❌
v2: 服装太现代 ❌
v3: 加入竹编元素 ✅

✅ 成果输出

最终提示词（祖父）：

Elderly Chinese craftsman, 70yo, gentle eyes with slight confusion, wearing traditional bamboo-blue work clothes, silver hair, weathered hands, warm lighting, photorealistic rendering
[角色定妆图]

分镜设计
规划24个镜头，确定视觉节奏
📥 设计输入

四幕时间分配：

• 第一幕：0-1.5min (镜头01-06)
• 第二幕：1.5-3.5min (镜头07-15)
• 第三幕：3.5-5min (镜头16-24)

色彩规划：

竹青 → 中国红 → 暖金

🤔 决策记录
24个镜头 5分钟片长 特写为主

关键镜头决策：

镜头18（中点）：必须展示"老地方"的揭晓
镜头22（高潮）：纸鸢飞起叠化年轻祖父
